🌌 OmniQuiz: Federated Slide Curation & Quiz Marketplace
OmniQuiz is an avant-garde, high-performance web engineering ecosystem built for collegiate quizzing circuits and academic network nodes. It operates as a dual-tenant digital asset clearing house, enabling verified publishers to monetize specialized slide presentations, MELA/TLC trivia matrices, and documentation bundles, while providing consumers with an interactive, user-gated discovery runway.

The architecture features a completely decoupled client/server boundary layout, persistent Base64 content pipeline streams, automated pricing modifiers, real-time yield diagnostics, and multi-tier usage authorization gates.

🎯 Strategic Goals & System Objectives
The platform replaces volatile, un-isolated media rendering techniques and static, broken file storage storage systems with an integrated, highly optimized framework designed to achieve four core engineering milestones:

Permanent SSR Hydration Fix: Complete decoupling of binary manipulation libraries (such as pdfjs-dist) from Node.js server-side pre-rendering phases. Canvas extraction processing tasks are securely wrapped inside client-isolated lifecycles, preventing compiler lockups.

Persistent Content Ingestion Streams: Moving away from volatile browser session blob: URLs that trigger garbage collection faults. Documents are parsed directly into immutable, non-volatile Base64 text streams stored right within your relational database indexing tables.

Automated Reputation & Multi-Factor Pricing Rules: Enforcement of platform monetary protocols where early creators (contributions ≤3) are bound to an unconditional free tier to build network authority. Post-reputation pricing unlocks dynamically based on algorithmic star ratings and genre multipliers.

Unique Account-Locked Metric Synchronization: Elimination of click-inflation loops and metric inflation. The platform evaluates logs via composite relational relational constraints ([userId,quizId]), tracking downloads and views strictly once per unique student profile.

🗂️ 1. Consumer Runway (Discovery Shelf)
Large Vertical Card Layouts: Content packages are displayed as spacious, modern vertical rectangular slots that stack cleanly down the viewport.

Page 1 Vector Poster Extraction: Utilizing browser canvas pipelines, the application decodes the Base64 data string, isolates index 1 of the PDF stream, and draws it as the card's live poster artwork.

Institutional Domain Matching: Reads incoming session email handles (e.g., @iitism.ac.in vs @iitb.ac.in) to instantly label cards with glowing neon university tracking badges (🏛️ IIT (ISM) DHANBAD vs 🏛️ IIT BOMBAY).

Community Plagiarism Regulation: Integrates an active dispute flag mechanism. Clicking "Flag Copy" toggles the backend database status, instantly converting the card’s emerald green seal into a warning red alert banner.

Self-Eviction Identity Safeguard: The red Delete Asset trigger compares your active user ID against the item's creator ID wrapper—rendering it functional and visible only to the user who deployed that specific set.

👤 2. Publisher Runway (Creator Studio)
Magical Coordinate Cursor Engine: Hides the native system arrow inside the studio boundary, tracking the mouse path with neon-purple canvas crosshairs (+) and a trailing cloud of glowing particle nodes.

Multi-Chunk Loading Wheel Ticker: Features an uncompressed drag-and-drop file target. Upload actions trigger a responsive loading percentage wheel running from 0% to 100%, sealing onto a verification badge once complete.

The Yield Diagnostics HUD Ledger: Tracks platform financial performance metrics (Gross Yield, Net Creator Wallet at 70%, Views, and Downloads). Employs the Zero-State Guard Rule: if verified uploads equal zero, the entire tracker is completely hidden and replaced with an idle placeholder card.

Multi-Bundle Combo Compiler: Once a creator has more than two active listings on the grid index, a box-checking matrix unseals, enabling them to compile multiple sets into a single package.

⚙️ 3. Universal System Layers & Localization
AI-Powered History Component: Positioned right underneath the main feed sits the "On This Day in Circuit History" terminal. Upon page hydration, an internal randomizer fetches a real historical event matching today's date and types it out via an animated typewriter sequence.

Instant Language Toggle Matrix: A prominent navigation link allowing users to toggle entire text string interfaces between clean English prose and clear Hindi translation arrays (en vs hi).

Neomorphic Theme Transformation: Shifts the workspace environment from the default deep cyber obsidian dark mode into a crisp, creamy white, soft-shadowed alternative theme layout instantly.

The Standalone Explore Index Hub: Located at the top-right navbar tracking link, clicking EXPLORE CIRCUIT INDEX ➔ shifts the background layout color schema dynamically and reveals automated, horizontal carousel scroll shelves tracking recently uploaded, recently viewed, and exclusively curated recommendations.
